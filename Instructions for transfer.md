# Инструкция по использованию приложения

## Адрес приложения
После запуска приложения, пожалуйста, перейдите по следующему адресу для входа:
[http://localhost:8080/login](http://localhost:8080/login)

## Примеры данных для переводов

Примеры переводов для пользователя "1@ps.ru":

| Параметр             | Значение                               | Описание                                     |
| :------------------- | :------------------------------------- | :------------------------------------------- |
| **Логин**            | `1@ps.ru`                                | Email пользователя для входа в систему       |
| **Пароль**           | `123`                                  | Пароль пользователя                          |

### Пример перевода клиенту с ID=6 (Ольга Попова)

**Сценарий 1: Перевод по номеру телефона**
Пользователь `1@ps.ru` может совершить перевод клиенту `Ольга Попова` по её номеру телефона.

- **Номер телефона получателя:** `+79166789012`

**Сценарий 2: Перевод по номеру карты**
Пользователь `1@ps.ru` может совершить перевод клиенту `Ольга Попова` на одну из её карт.

- **Номер карты получателя:** `100011` или `100012`

**Данные клиента для перевода:**

| Параметр             | Значение               |
| :------------------- | :--------------------- |
| **Имя**              | `Olga`                 |
| **Фамилия**          | `Popova`               |
| **Номер телефона**   | `+79166789012`         |
| **Карты**            | `100011`, `100012`     |
